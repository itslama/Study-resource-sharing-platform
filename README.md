# Study-resource-sharing-platform
A theoretical software engineering project developed for CS 385, proposing a Study Resources Sharing Platform where university students can collaborate by sharing notes, summaries, presentations, e-books, and past exams. The system is designed around an Incremental development model, delivering features step by step.
The project covers the full software engineering documentation lifecycle, including functional and non-functional requirements, UML diagrams, and use case scenarios.
Key Features (Proposed): User registration & authentication · Resource upload/download (PDF, DOCX, images) · Keyword search & filtering · Rating & commenting · Notification system · Admin content moderation
Deliverables: System description · Requirements (functional & non-functional) · Context diagram ·Use case diagram & scenarios · Class diagram · Sequence diagram · Activity diagrams
